Чтобы после неудачного ввода продолжить работу с потоком, нужно сбросить состояние ошибки, вызвав метод потока `clear()`. После этого из потока можно возобновить чтение:

```cpp
int n;
if (std::cin >> n) {
    std::cout << "Введено число " << n << std::endl;
} else {
    // Только после clear можно продолжить читать из потока.
    std::cin.clear();
    
    std::string word;
    if (std::cin >> word) {
        std::cerr << "Вы ввели " << word << " вместо числа" << std::endl;
    } else {
        std::cerr << "Не удалось ничего прочитать из потока" << std::endl;
    }
}
```
