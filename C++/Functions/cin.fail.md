Чтобы определить, успешно ли завершилось чтение данного типа из потока, используют метод `fail()`. Этот метод возвращает `true`, если последняя операция чтения из потока завершилась неудачей. После этого продолжать чтение из потока нельзя. Рассмотрим пример:

```cpp
int n;
std::cin >> n;
if (!std::cin.fail()) {
    std::cout << "Введено число " << n << std::endl;
} else {
    // Вместо std::cout используем std::cerr.
    // Это поток, предназначенный специально для сообщений
    // об ошибках.
    std::cerr << "Ошибка при вводе числа" << std::endl;
}
```

Если вместо числа ввести в поток строку `abc`, то программа сообщит об ошибке. Для сообщений об ошибках, как правило, используют специальный поток — `std::cerr`. Работает он так же, как поток `std::cout`. Обычно с его помощью отделяют рабочий вывод программы от технических сообщений об ошибках, если они возникают.

Сам поток может быть конвертирован к `bool`, при этом результат будет аналогичен вызову `fail()` и его отрицанию:

```cpp
int n;
std::cin >> n;
if (std::cin) {
    std::cout << "Введено число " << n << std::endl;
} else {
    std::cerr << "Ошибка при вводе числа" << std::endl;
}
```

Можно ещё упростить эту запись, воспользовавшись тем, что результат операции `>>` — это сам поток:

```cpp
int n;
if (std::cin >> n) {
    std::cout << "Введено число " << n << std::endl;
} else {
    std::cerr << "Ошибка при вводе числа" << std::endl;
}
```

Чтобы после неудачного ввода продолжить работу с потоком, нужно сбросить состояние ошибки, вызвав метод потока [clear](cin.clear). После этого из потока можно возобновить чтение:

```cpp
int n;
if (std::cin >> n) {
    std::cout << "Введено число " << n << std::endl;
} else {
    // Только после clear можно продолжить читать из потока.
    std::cin.clear();
    
    std::string word;
    if (std::cin >> word) {
        std::cerr << "Вы ввели " << word << " вместо числа" << std::endl;
    } else {
        std::cerr << "Не удалось ничего прочитать из потока" << std::endl;
    }
}
```
