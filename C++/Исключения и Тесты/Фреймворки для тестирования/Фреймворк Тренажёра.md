В практической части урока вы познакомитесь с ещё одним фреймворком, созданным специально для решения задач в Тренажёре. Он работает аналогично «большим» фреймворкам (вроде Google Test и Catch2). Вот как можно его использовать:

```
#include "practicum_tests.h"
#include "file_with_code_to_test.h" // Подключение кода, который будет тестироваться.

PRAC_TEST(TestName) {
    PracCheck(<условие для проверки>, <комментарий>);
} 
```

Для объявления теста нужен макрос `PRAC_TEST`, в скобках указывается название теста, которое должно быть уникальным. Для проверки используется команда `PracCheck`, которой передаются логическое выражение и комментарий с описанием, что проверяет этот тест.
