Если вы хотите увидеть не просто лог, а конкретные изменения в коммите, используйте команду `git show`. Введите эту команду, а после неё укажите хеш коммита, который вас интересует. Для удобства можно использовать только первые семь символов идентификатора коммита.

```sh
$ git show ab17582

# Информация о коммите.
commit ab17582464548e807d38996d37aee28914377321
Author: User <user@yandex.ru>
Date:   Mon Oct 23 10:26:48 2023 +0300

    Добавлена закрывающая скобка

diff --git a/main.cpp b/main.cpp
index 4988dd2..741b76d 100644
--- a/main.cpp
+++ b/main.cpp
@@ -2,3 +2,4 @@

# Исходные строки в изменённом файле.
 int main() {
     std::cout << "Hello, Git!" << std::endl;
# Строка с изменениями.
+} 
```

Чтобы посмотреть информацию о последнем коммите, выполните команду `git show` без указания идентификатора.

```sh
$ git show

commit d9b48163447db6da8f31447693469f84a2cb1503 (HEAD -> master, origin/master, origin/HEAD)
Author: User <user@yandex.ru>
Date:   Mon Oct 23 10:46:17 2023 +0300

    Удалён файл bad.cpp

diff --git a/bad.cpp b/bad.cpp
deleted file mode 100644
index e69de29..0000000 
```
