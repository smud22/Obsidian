Файл bad.cpp — лишний файл в удалённом репозитории. Чтобы его не отслеживать, выполните команды:

```sh
# Удалить файл из отслеживания, но оставить на компьютере.
git rm --cached bad.cpp

# Зафиксировать изменения в локальном репозитории.
git commit -m "Удалён файл bad.cpp"

# Отправить изменения в удалённый репозиторий.
git push 
```

Файл удалён из репозитория на GitHub (можете убедиться в этом), и локальные изменения в нём больше не отслеживаются. Теперь можно либо удалить его из директории с проектом, либо добавить в .gitignore, если файл должен остаться в локальном репозитории.
