Если после создания коммита вы хотите внести ещё одну маленькую правку, можно не делать новый коммит. Для этого есть ключ `--amend` (от англ. amend — «исправить»):

```sh
git commit --amend -m "Текст вашего сообщения" 
```

Эта команда добавит изменённые файлы в ваш последний коммит, а с дополнительным флагом `-m` ещё и обновит сообщение:

```sh
# Делаем первый коммит, в кавычках пишем сообщение.
git commit -m "Исправлен файл main.cpp"

# Добавляем все изменения в индекс Git.
git add .

# Добавляем эти изменения к предыдущему коммиту.
git commit --amend -m "Исправлен main.cpp и добавлены новые файлы" 
```
