Функция может извлекать информацию из структуры и возвращать значение другого типа. Например, так можно вычислить яркость цвета:

```cpp
float GetBrightness(RGBAColor c) {
    float brightness = 0.299f * c.red + 0.587f * c.green + 0.114f * c.blue;
    return brightness;
} 
```

Не все компоненты цвета вносят одинаковый вклад в видимую глазом яркость. Поэтому пришлось использовать коэффициенты 0.299, 0.587, 0.114. Эти коэффициенты рекомендованы международным стандартом для распространённого цветового пространства Rec. 709 RGB.
