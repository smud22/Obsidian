Константные методы - методы, не изменяющие сам вектор

Неконстантные методы - методы, изменяющие при своем вызове значения в векторе

Методы `front` и `back` могут быть как константными, так и неконстантными, так как они позволяют изменить элемент при необходимости.

```cpp
// front и back возвращают изменяемое значение — l-value.
// Оно может использоваться слева от знака равенства.
numbers.front() = 0;

// Это касается и операции [].
numbers[3] = 123; 
```
