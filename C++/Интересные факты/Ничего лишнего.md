В условии для конструкции `if`-`else` можно использовать переменные из внешней области видимости, а в блоках кода — внешние и локальные переменные. Но есть и третий вариант: прямо внутри условия можно объявить переменную, которую можно использовать в этом условии, а также внутри `if` и `else`. Вне оператора `if` и всех зависимых от него `else if` и `else` переменная не будет видна.

```cpp
if (int x; std::cin >> x) {
    std::cout << "Вы ввели число " << x << std::endl;
} else {
    std::cout << "Ошибка ввода" << std::endl;
}

// Тут переменная x не видна.
```
