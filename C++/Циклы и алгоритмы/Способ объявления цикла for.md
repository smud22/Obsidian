Заметьте, что в начале цикла нужно написать переменную `i` трижды — при этом легко ошибиться. В современном C++ есть более удобный вариант такого же цикла:

```cpp
for (int i : std::views::iota(1, 101)) {
    std::cout << i << std::endl;
} 
```

В `iota` указывается первое число (включительно) и последнее (не включительно). Для работы этой функции нужно добавить `#include <ranges>`. Этот способ работает только с достаточно современными компиляторами.
