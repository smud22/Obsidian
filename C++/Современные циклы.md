#include <ranges>

// Также вынесем метод в .cpp-файл.
void MainWindow::paintEvent(QPaintEvent *) {
    QPainter painter(this);
    painter.setPen(Qt::NoPen);

    // Используем два вложенных цикла со счётчиком от 0 до 7.
    for (int i : std::views::iota(0, 8)) {
        for (int j : std::views::iota(0, 8)) {
            // Рисуем чёрный либо белый прямоугольник.
            // Чередуем их в шахматном порядке.
            bool is_black = (i + j) % 2 != 0;
            if (is_black) {
                painter.setBrush(Qt::black);
            } else {
                painter.setBrush(Qt::white);
            }
            
            // Прямоугольник 50х50. Отступы от края 
            // формы зависят от i и j.
            painter.drawRect(100 + i * 50, 
                             100 + j * 50, 
                             50, 50);
        }
    }
}

`std::views::iota`

 (from C++20 Ranges) creates a lazy, infinite or bounded sequence of incrementing values (like `0, 1, 2, ...`), acting as a powerful range factory that generates numbers on demand, unlike the older `std::iota` algorithm which fills a pre-existing container. It's highly composable with other range adaptors, allowing operations like `views::iota(0, 10) | views::filter(...)` to create filtered sequences without temporary storage. 

Key Characteristics

- **Lazy Evaluation:** Generates numbers only when iterated over, making it memory-efficient for large or infinite sequences.
- **Bounded & Unbounded:** Can create an infinite sequence (e.g., `views::iota(0)`) or a bounded one (e.g., `views::iota(0, 10)`).
