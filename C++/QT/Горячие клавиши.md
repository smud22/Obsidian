| Клавиша Windows       | Клавиша macOS       | Действие                                                                                                                                                                                                          |
| --------------------- | ------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| F4                    | F4                  | Переключение между файлом реализации (.срр) и заголовочным файлом (.h)                                                                                                                                            |
| F2 (либо Ctrl-клик)   | F2 (либо ⌘-клик)    | Переход к определению выбранной функции или типа                                                                                                                                                                  |
| Ctrl-{1..6}           | Ctrl-{1..6}         | Переключение между вкладками слева (1 — Начало, 2 — Редактор, 3 — Дизайн, 4 — Отладка, 5 — Проекты, 6 — Справка)                                                                                                  |
| Alt-{1..8}            | ⌘{1..8}             | Переключение между вкладками снизу (1 — Проблемы, 2 — Результаты поиска, 3 — Вывод приложения, 4 — Вывод сборки, 5 — Консоль отладчика, 6 — Основные сообщения, 7 — Контроль версий, 8 — Результаты тестирования) |
| Ctrl-/                | ⌘/                  | Комментирование выделенного кода                                                                                                                                                                                  |
| Ctrl-I                | ⌘;                  | Автоматическое форматирование выделенного участка кода. Чтобы выделить весь код в файле, используйте Ctrl-A                                                                                                       |
| Alt-влево, Alt-вправо | ⌥⌘-влево, ⌥⌘-вправо | Переход по коду назад и вперёд                                                                                                                                                                                    |
