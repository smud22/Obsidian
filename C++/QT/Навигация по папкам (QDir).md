Изучим ещё один класс, который называется `QDir`, где Dir — это сокращение от слова directory (англ. «директория»). Этот класс полезен, если вашей программе необходимо взаимодействовать с файловой системой. Например, обратиться к заданной папке, прочитать её содержимое, найти необходимый файл.

`QDir` может читать не только файлы диска, но и папки файла ресурсов. Эта особенность пригодится при решении задачи.

При создании объекта с типом `QDir` в созданной переменной (например, с именем dir) в круглых скобках указывают путь до папки:

```cpp
// Подключить библиотеку класса.
#include <QDir>
...
// Создать объект с типом QDir. 
QDir dir("путь до папки"); 
```

Чтобы получить содержимое папки, используйте метод `entryList`. Этот метод возвращает тип `QStringList`, то есть список строк `QString`. Применив метод `size()` к возвращённому списку, вы узнаете количество элементов в папке. Так как переменная `dir` является обычной переменной, а не указателем, то обращение к объекту с типом `QDir` будет выполняться через точку:

```cpp
QDir dir("путь до папки");
QStringList list = dir.entryList();
int num_objects = list.size(); 
```

Так как `QStringList` является списком, то пройтись по его элементам можно либо в цикле, либо обратившись по индексу к одному из элементов. Но метод `entryList` возвращает только имена объектов. Если, к примеру, в вашей папке содержатся файл main.cpp и папка «Первый спринт», то вы получите следующий набор элементов:
```sh
.
..
main.cpp
Первый спринт 
```

Обратите внимание, что в список, который вернул `entryList`, попали лишь названия файлов. Чтобы из названия файла получить полный (абсолютный) путь, используйте метод `filePath`.

```cpp
QDir dir("путь до папки");
QStringList list = dir.entryList();

if(list.size() > 2) {
  // Пропустим первые два элемента 
  // и прочитаем сразу третий (с индексом 2).
  QString path = dir.filePath(list[2]);
} 
```

Чтобы получить подробную информацию об объектах, которые расположены в папке, используйте другой метод получения списка `entryInfoList()`. Он выдаёт объекты с типом `QFileInfo`. Элементы, которые содержатся в папке, могут быть файлами или вложенными папками. Для проверки типа объекта используется метод `isFile()`, который вернёт true, если элемент является файлом. Чтобы проверить, является ли элемент директорией, существует метод `isDir()`.

Более подробно про библиотеку класса `QDir` вы можете почитать [в документации Qt](https://doc.qt.io/qt-6/qdir.html).
