Название функции записывается в CamelCase («верблюжьем регистре») — каждое слово пишется с заглавной буквы.

```cpp
void HideAndSeek() {
}
```

В круглых скобках мы добавили параметр `std::string name`. Для указания параметра требуется указать его тип (в нашем случае это строка `std::string`) и название параметра. Параметр — это переменная, в которой будет храниться то значение, что передали снаружи функции. Время жизни параметра совпадает со временем работы функции, и к параметру можно обратиться в любом месте этой функции. Пока функция не вызвана, эта переменная не имеет значения. Такие параметры называются **формальными**.

правила именования формальных параметров: названия параметров будем записывать, по аналогии с обычными переменными, в snake_case («змеином регистре») — каждое слово пишется маленькими буквами, и слова разделяются знаком `_`.
