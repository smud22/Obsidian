Для перегрузки функций в качевте аргументов могут быть заданы параметры, которые вовсе не будут использоваться при ее вызове

```cpp
std::string TypeToString([[maybe_unused]] int arg) {

    return "int";
}
```

Конструкция `[[maybe_unused]]` нужна, чтобы избежать предупреждения о неиспользуемом аргументе. Вместо неё можно просто не писать имя аргумента: `std::string TypeToString(int)`, но такой способ считается менее современным.
