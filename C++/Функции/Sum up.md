Функции — это строительные блоки, используя которые вы можете создавать большие программы. Ещё функции помогают устранить дублирование кода, улучшить читаемость программ и облегчить их тестирование.

Вы узнали несколько способов передать аргументы в функцию. Стандартный способ — передача по значению. Передача по ссылке используется, чтобы изменение аргумента внутри функции было видно вызывающей стороне. Передача аргумента по константной ссылке используется как альтернатива передаче по значению, если копирование аргумента накладно.

Перегрузка функций позволяет определить несколько функций с одинаковым именем, но разными аргументами. Это сокращает количество различных имён в программе и облегчает использование однотипных функций. При вызове функции компилятор выбирает из кандидатов ту, которая лучше всего подходит по количеству и типам аргументов.

В многофайловых программах совместно используемые функции объявляют в заголовочном файле, а определяют в соответствующем ему .cpp-файле. Такое разбиение программы на файлы помогает структурировать код и упрощает её поддержку.
