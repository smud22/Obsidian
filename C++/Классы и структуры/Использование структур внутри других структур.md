Объявив структуру, её можно использовать не только для объявления переменных, но и в качестве полей других структур. Это можно сравнить с тем, как из элементарных частиц составляются атомы, из атомов — молекулы, а из молекул — простые и сложные организмы.

Здесь структура `Date` используется внутри структуры `Person` для хранения даты рождения.

```cpp
struct Date {
    int year;
    int month;
    int day;
};

struct Person {
    std::string name;
    Date birth_date;
}; 
```

Старайтесь по максимуму использовать ранее написанный и протестированный код. Это снижает количество ошибок и сокращает время разработки. Начинающие разработчики часто пишут новый код, а не используют существующий. Такой код в шутку называют велосипедом.

Поля структурного типа инициализируются так же, как и переменные-структуры.

Рассмотрим на примерах.

```cpp
int main() {
    using namespace std::literals;

    Person p1{"Leo Tolstoy"s, { 1828, 9, 9 }};
    
    // Можно явно указать тип вложенной структуры.
    auto p2 = Person{"Alexander Pushkin"s, Date{1799, 6, 6}};

    // Вложенные структуры тоже могут использовать назначенную инициализацию.
    Person p3 {
        .name = "Nikolai Gogol"s,
        .birth_date = { .year = 1809, .month = 4, .day = 1 }
    };

    // Если не указать значение вложенной структуры,
    // она будет проинициализирована значением по умолчанию.
    Person p4 { "John Doe" }; // Дата рождения: 0000.00.00.
} 
```

Для доступа к полям вложенной структуры может понадобиться более одной точки.

```cpp
int main() {
    using namespace std::literals;

    Person person;
    // Обращаемся к полю структуры Person.
    person.name = "Ivan Smirnov";

    // Обращаемся к полям структуры Date внутри структуры Person.
    person.birth_date.year = 2000;
    person.birth_date.month = 11;
    person.birth_date.day = 30;
} 
```
